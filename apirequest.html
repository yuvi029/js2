XMLHttpRequest (XHR) is a JavaScript object used to interact with servers — it allows you to send HTTP requests and receive data without reloading the web page.

readyState State Name Description

  • UNSENT The XMLHttpRequest object is created, but the open() method hasn’t been called yet.
  • OPENED The open() method has been called — the request is set up but not sent.
  • HEADERS_RECEIVED send() has been called, and the response headers have been received.
  • LOADING The response body is being received (data is being downloaded).
  • DONE The operation is complete. You can now access responseText, status, etc.

  • GET Fetch data POST Send new data PUT Update data DELETE Remove data
  • V8 ENGINEThe V8 engine is Google’s open-source JavaScript engine used to execute JavaScript code. It was originally built for Google Chrome, but now powers other platforms too — like Node.js.
